# 🌸 Liora

### Your Personal AI Bedroom Stylist

Liora is an AI-powered bedroom styling application that helps users transform their bedrooms into spaces that better reflect their personality and desired aesthetic.

Unlike traditional AI room generators, Liora **does not create unrealistic redesigned images.**

Instead, it analyzes a user's existing bedroom and provides thoughtful styling recommendations while preserving the room's unique character.

---

## ✨ Features

- 🛏️ Bedroom image analysis
- 🌸 Personalized styling recommendations
- 🎨 Bedroom personality detection
- 💡 Lighting suggestions
- 🪴 Decor & greenery recommendations
- 🖼️ Wall styling ideas
- 🧺 Organization suggestions
- 🎀 Dream Bedroom Vision
- 💌 Personalized letter from Liora
